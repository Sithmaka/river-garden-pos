# Print Sales Reports - Quick User Guide

## ✨ New Feature: Print Sales Reports

You can now print professional sales reports directly from the admin dashboard!

---

## 🎯 What You Can Print

### Reports Available

- **Daily Report** - Last 7 days of sales
- **Weekly Report** - Last 4 weeks of sales
- **Monthly Report** - Last 12 months of sales

### What's Included

✅ All items sold in the period
✅ Quantity sold per item
✅ Revenue per item
✅ Total sales amount
✅ Service charge details
✅ Professional receipt format

---

## 📖 How to Print a Report

### Step-by-Step Guide

**Step 1: Select a Time Period**

```
On the dashboard, click one of these buttons:
[ Daily ] [ Weekly ] [ Monthly ]
```

The dashboard will show data for that period.

**Step 2: Review the Data**

- Check the sales chart
- Review the statistics
- See the top-selling items
- Verify service charge information

**Step 3: Click Print Report**

```
Look for the green button:
🖨️ Print Report
(top-right of the Sales Summary section)
```

**Step 4: Preview the Receipt**
A preview window will open showing:

- How the receipt will look
- All the data that will print
- Scrollable view for long reports

**Step 5: Choose Your Action**

```
Two buttons appear:
[Cancel]          [🖨️ Print Now]
Close without     Send to printer
printing
```

**Step 6: Select Printer (if you click Print Now)**

```
Browser print dialog appears:
1. Select your printer from the dropdown
2. Adjust settings if needed (usually already correct)
3. Click "Print" to print the report
```

**Step 7: Collect Your Receipt**
The receipt prints to your thermal printer in seconds!

## 💡 Tips & Tricks

### 🔥 Pro Tips

**Tip 1: Review Before Printing**
Always check the preview before clicking "Print Now"

**Tip 2: Different Reports for Different Purposes**

- **Daily**: For end-of-day reconciliation
- **Weekly**: For weekly staff meeting
- **Monthly**: For monthly financial reporting

**Tip 3: Print Multiple Reports**
You can print each period separately:

1. Click Daily → Print → Click Print Now
2. Click Weekly → Print → Click Print Now
3. Click Monthly → Print → Click Print Now

**Tip 4: Save as PDF**
In the print dialog, instead of selecting a printer, choose "Print to PDF" or "Save as PDF" to create a PDF file instead of printing

**Tip 5: Check Your Data First**
If the report looks wrong, check that you selected the correct time period

## ⚙️ Printer Setup

### Which Printers Work?

The print feature works with ANY printer:

- ✅ Thermal receipt printers (XPrinter, etc.)
- ✅ Inkjet printers
- ✅ Laser printers
- ✅ PDF printers (Save as PDF)

### Printer Recommended

- **Xprinter XP-80C** (thermal 72mm) - Best for receipts
- Other 72mm thermal printers - Also very good
- Any network printer - Works fine

### Print Settings

Usually the browser sets these automatically:

- Paper Size: 72mm (for thermal) or Auto
- Margins: None or Minimal
- Orientation: Portrait
- Scale: Fit to page

## ❌ Common Issues & Solutions

### Issue: Button is Grayed Out (Disabled)

**Problem**: The green "🖨️ Print Report" button looks faded

**Causes**:

- ⏳ Data is still loading (wait for the spinner to finish)
- 📊 No sales data for this period (try a different period)
- 🔄 System is preparing the report (wait a moment)

**Solution**:

1. Make sure data has loaded completely
2. Try selecting a different time period that has sales
3. Click the button again after a few seconds

---

### Issue: Preview Doesn't Show

**Problem**: Click Print Report but nothing happens

**Causes**:

- ⚠️ Technical error occurred
- 📡 Connection issue with database
- 🔌 Browser plugin blocking preview

**Solution**:

1. Check browser console for error (F12 key)
2. Verify you have sales data
3. Refresh the page and try again
4. Try a different time period

---

### Issue: Print Dialog Doesn't Open

**Problem**: Click "Print Now" but nothing happens

**Causes**:

- 🔌 Pop-up blocker is active
- 🖨️ No printers configured in browser
- ⚙️ Browser security settings

**Solution**:

1. Allow pop-ups from this website in your browser
2. Configure a printer in your browser/OS
3. Check browser security settings
4. Try a different browser

---

### Issue: Printed Receipt Looks Wrong

**Problem**: Text is cut off, spacing is bad, or format is wrong

**Causes**:

- 🖨️ Wrong printer selected
- ⚙️ Print settings not optimal
- 📄 Wrong paper size selected

**Solution**:

1. Use recommended printer settings
2. Check "Fit to page" is selected
3. Verify 72mm paper width (for thermal)
4. Try printing to PDF first to preview

---

### Issue: Report Shows Wrong Data

**Problem**: Items or totals don't match what you expect

**Causes**:

- 🗓️ Wrong time period selected
- 📊 Data not updated yet
- 🔄 Dashboard data out of sync

**Solution**:

1. Double-check the selected period (Daily/Weekly/Monthly)
2. Refresh the dashboard page
3. Check if new sales were entered today
4. Verify data in the main sales view

## ❓ FAQ

**Q: Can I edit the report before printing?**
A: The report contents come directly from sales data. To change what prints, check your sales records.

**Q: Can I print a custom date range?**
A: Not in this version. Currently offers Daily (7 days), Weekly (4 weeks), or Monthly (12 months).

**Q: Can I email the report?**
A: Not directly. But you can save as PDF and email it manually.

**Q: How many pages will it print?**
A: Usually 1-2 thermal rolls depending on how many items sold.

**Q: Can I print multiple times?**
A: Yes! You can print the same report as many times as you want.

**Q: Does it print customer details?**
A: No, only aggregated sales data (items, quantities, totals).

**Q: What if there's no data?**
A: The print button will be disabled and you'll see a message.

**Q: Can I customize the report format?**
A: For now, no. This is the standard format. Customization coming in future versions.

**Q: Do I need to be online?**
A: Yes, you need internet to load sales data from the database.
